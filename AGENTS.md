# ガイドライン

## レポジトリ

- GitHubのf0reachARR/wifimanに置かれています

## コマンド

- pnpmを使用して依存関係を管理してください。

## TypeScriptの使用

- TypeScriptのstrictモードを有効にしてください。
- 可能な限り型注釈を使用してください。
- シングルクォートとセミコロンを使用してください。
- コードの品質を保つために、Biomeを使用してコードをフォーマットし、tscを使用して型チェックを行ってください。

## Drizzleのマイグレーション作成

Drizzleのマイグレーションを作成する際は、以下の手順に従ってください。

- docker compose up -d postgres を実行してPostgreSQLコンテナが起動していることを確認してください。
- pnpm db:generate コマンドを実行して、マイグレーションファイルを生成してください。
- 生成されたマイグレーションファイル(./apps/api/drizzle以下にあるはずです)を確認し、必要に応じて編集してください。
- pnpm db:migrate コマンドを実行して、マイグレーションを適用してください。

## 一時ファイル

- 一時ファイルは .github/tmp ディレクトリに保存してください。
